# server-manager

To install dependencies:

```bash
bun install
```

To run:

```bash
bun index.ts
```

To compile:

```bash
bun run compile
```

.env:

```bash
CS2_PATH= #ex: C:\steamcmd\cs2-ds
STEAMCMD_PATH= #ex: C:\steamcmd
ACCESS_KEY= #abc
PORT=5817
CS2_SERVER_ARGS= #optional

```
